Q: How can I get a technical support for Maximo-CPI?
A: Due to limited resources, IBM Maximo Cluster Performance Insights is currently provided on an "AS IS" basis. However, You can report defects and feature requests at GitHub Issues. For performance diagnostics, you can upload a JSON file to your support ticket for further analysis.

Q: How do I know the current version of Maximo-CPI?
A: Refer to the latest version in /opt/app-root/src/CHANGELOG.md inside the container. If CHANGELOG.md does not exist, it is v1.0
